# Read the log file (`log-ablation.Rdata`) produced by [`ablation()`](https://mlopez-ibanez.github.io/irace/reference/ablation.md).

Read the log file (`log-ablation.Rdata`) produced by
[`ablation()`](https://mlopez-ibanez.github.io/irace/reference/ablation.md).

## Usage

``` r
read_ablogfile(filename)
```

## Arguments

- filename:

  `character(1)`  
  Filename that contains the log file saved by
  [`ablation()`](https://mlopez-ibanez.github.io/irace/reference/ablation.md).
  Example: `log-ablation.Rdata`.

## Value

[`list()`](https://rdrr.io/r/base/list.html)
