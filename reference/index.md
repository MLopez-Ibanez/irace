# Package index

## Launching irace

- [`irace()`](https://mlopez-ibanez.github.io/irace/reference/irace.md)
  : Execute one run of the Iterated Racing algorithm.

- [`irace_cmdline()`](https://mlopez-ibanez.github.io/irace/reference/irace_cmdline.md)
  [`irace.cmdline()`](https://mlopez-ibanez.github.io/irace/reference/irace_cmdline.md)
  :

  Launch `irace` with command-line options.

- [`irace_main()`](https://mlopez-ibanez.github.io/irace/reference/irace_main.md)
  : Higher-level interface to launch irace.

- [`multi_irace()`](https://mlopez-ibanez.github.io/irace/reference/multi_irace.md)
  :

  Execute
  [`irace()`](https://mlopez-ibanez.github.io/irace/reference/irace.md)
  multiple times with the same or different scenarios and parameter
  space definitions.

- [`testing_fromfile()`](https://mlopez-ibanez.github.io/irace/reference/testing_fromfile.md)
  : Test configurations given an explicit table of configurations and a
  scenario file

- [`testing_fromlog()`](https://mlopez-ibanez.github.io/irace/reference/testing_fromlog.md)
  :

  Test configurations given in the logfile (typically `irace.Rdata`)
  produced by irace.

## Analysis

- [`getConfigurationById()`](https://mlopez-ibanez.github.io/irace/reference/getConfigurationById.md)
  : Returns the configurations selected by ID.

- [`getConfigurationByIteration()`](https://mlopez-ibanez.github.io/irace/reference/getConfigurationByIteration.md)
  : Returns the configurations by the iteration in which they were
  executed.

- [`getFinalElites()`](https://mlopez-ibanez.github.io/irace/reference/getFinalElites.md)
  : Return the elite configurations of the final iteration.

- [`get_instanceID_seed_pairs()`](https://mlopez-ibanez.github.io/irace/reference/get_instanceID_seed_pairs.md)
  : Returns the pairs of instance IDs and seeds used as instances in the
  race (and optionally the actual instances).

- [`irace_summarise()`](https://mlopez-ibanez.github.io/irace/reference/irace_summarise.md)
  : Summarise the results of a run of irace

- [`read_logfile()`](https://mlopez-ibanez.github.io/irace/reference/read_logfile.md)
  :

  Read the log file produced by irace (`irace.Rdata`).

- [`save_irace_logfile()`](https://mlopez-ibanez.github.io/irace/reference/save_irace_logfile.md)
  :

  Save the log generated by irace to a file (by default `irace.Rdata`).

## Ablation

- [`ablation()`](https://mlopez-ibanez.github.io/irace/reference/ablation.md)
  : Performs ablation between two configurations (from source to
  target).

- [`ablation_cmdline()`](https://mlopez-ibanez.github.io/irace/reference/ablation_cmdline.md)
  : Launch ablation with command-line options.

- [`plotAblation()`](https://mlopez-ibanez.github.io/irace/reference/plotAblation.md)
  : Create plot from an ablation log

- [`read_ablogfile()`](https://mlopez-ibanez.github.io/irace/reference/read_ablogfile.md)
  :

  Read the log file (`log-ablation.Rdata`) produced by
  [`ablation()`](https://mlopez-ibanez.github.io/irace/reference/ablation.md).
